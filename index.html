Kubernetes in Qemu

Introduction

The following excercise will build a full kubernetes cluster installation on top of an heterogeneous set of qemu virtual machines as worker nodes.

Steps:

  1. Prepare the host and virtual network
  2. Create the virtual machines
  3. Install k8s worker and create a control plane
  4. Join worker vms to the cluster

Each worker will be a different Virtual Machine running Debian GNU/Linux on Qemu to show node affinities.
