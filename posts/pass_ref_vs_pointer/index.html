Pass by reference is not passing a *pointer*

August 15, 2025

Passing by reference is NOT the same as passing a POINTER

in Java:

theObj method(theObj)

public void method(obj MyObject){ obj.name = “blah”; obj = null; // This doesn’t change theObj cause obj is a pointer is not a reference! }

This can be seen in Ada/Fortran/Pascal

subroutine swap(a, b) integer :: a, b integer :: tmp tmp = a a = b b = tmp end subroutine
