Cloud images

August 15, 2025

Create a cloud-init seed image

Download the debian latest PlainVM cloud image: debian-XX-genericcloud-amd64.qcow2

hostname: vm${i}
ssh_authorized_keys:
  - ssh-rsa AAAA..... user@host

And generate the VM seed

cloud-localds vm${i}.seed vm${i}.cfg


qemu-img create -f qcow2 -b debian-XX-genericcloud-amd64.qcow2 -F qcow2 vm${i}.img

create an overlay qcow2

Run the VM’s

  qemu-system-x86_64 \
    -m 2048 \
    -smp 2 \
    -enable-kvm \
    -drive file=vm0.img,if=virtio \
    -drive file=vm0.seed,if=virtio,format=raw,media=cdrom \
    -netdev tap,id=net0,ifname=tap0,script=no,downscript=no \
    -device virtio-net-pci,netdev=net0,romfile="",mac=52:54:00:00:00:01 \
    -display default \
    -serial unix:/tmp/vm0.sock,server,nowait \
    -name "vm0"
