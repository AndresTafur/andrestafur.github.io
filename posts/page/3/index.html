Monotonic clocks

Wall clock vs. Monotonic clock

Wall clock: the “real world” time (time.Now()), which can jump forward/backward if the system time is adjusted (e.g. NTP, manual clock change, daylight saving).

Monotonic clock: a clock that only moves forward at a constant rate. It’s not affected by system time changes. It’s ideal for measuring elapsed time.

Untyped constants

An untyped constant is pretty much a literal; It doesnt have a type of its own altough it haves a default type that is used when no other type can be inferred.

For example this works:
