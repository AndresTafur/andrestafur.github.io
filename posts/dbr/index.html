Cross cloud DBR

Cross-cloud sync involves:

Replication: Copying data from one cloud to another.

Synchronization: Keeping changes consistent (two-way or one-way).

Transformation: Converting data formats or schemas between systems.

Conflict resolution: Handling simultaneous updates in different clouds.

Latency management: Ensuring timely propagation while avoiding overload.
