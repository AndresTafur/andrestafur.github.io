Literals vs immutability

August 15, 2025

Immutability and constants

A constant var is a variable that is known at compile-time, in java it is defined as ‘static final’, in golang it is defined with ‘const’.

On the other hand Immutability is about not modifying a variable even if it is created and assigned at run-time, in java it would be ‘final’ again but these two concepts are entirely different.
